采用设备像素进行开发

因为一般设计稿都是最大分辨率进行设计的，即基于设备像素进行设计，所以开发的时候也基于设备像素进行开发是比较方便的。通过 document.documentElement.clientWidth 可以拿到设备像素，设备像素 = css像素（理想适口宽度） * window.devicePixelRatio(设备像素比)
需要 js 动态获取当前设备的设备像素，设置 meta，进行缩放。这时，开发中 css 的像素就是针对设备像素而言的。css中的像素值可以直接根据设计稿来定义啦。

针对不同设备进行缩放

不同设备的具有不同的设备像素，如何将针对 iphone6 的设计稿也同样适用于 iphone5 呢？这里采用 rem 来解决这个问题。1rem 等于 html元素的 fontSize。我们针对不同的设备（iphone5, iphone6 等等）设置不同的fontSize = 设备分辨率的1/10。那么 CSS 中，1rem 就等于屏幕的1/10，就可以针对不同的设备宽度进行缩放了。

这里 ratio 的大小针对设计稿中的设备像素宽度决定，比如，设备像素宽度为750px，radio=7.5， 此时 1rem = 100px，定义宽度时就可以直接将设计稿中元素的宽度除以 100，如果设备像素为1200， radio= 12，1rem = 100px, 比较方便计算。
