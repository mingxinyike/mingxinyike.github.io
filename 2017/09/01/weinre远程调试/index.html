weinre 远程调试移动设备

想要解决的问题：调试移动端样式

weinre 是什么

weinre (Web Inspector Remote)是基于Web Inspector(Webkit)的远程调试工具。可以直接在电脑上调试运行在手机上的远程页面。

weinre 安装

运行 werine

安装成功会出现

在浏览器地址栏中输入下面的网址打开Weinre调试工具。

ip地址 –> 命令行 ifconfig
该网页中 Target script 要加载需要调试的页面中
该网页中的 Access Points debug client user interface: http://10.242.10.198:8082/client/#anonymous 点击进入出现调试页面

然后用手机连接调试页面，再调试页面中target列表中选中需要调试的URL，elements中可以看到样式
