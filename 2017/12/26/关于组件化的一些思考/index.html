关于组件化的一些思考

什么是组件化

组件化就是从设计的层面上对 UI 进行拆分，从代码层面进行封装的过程。

组件化和模块化的区别

模块化是代码层面的拆分，而组件化是在模块化的基础上，从设计层面对 UI 进行拆分。

组件可以分为哪几类

根据组件的通用性（复用性），可以分为三类：

  • 通用组件 （不同产品间可以复用的组件）
  • 通用业务组件 （同一产品中可以复用的组件）
  • 业务组件 （不可复用的组件）
