NEJ template

document.createDocumentFragment
返回一个文档片段，对该文档片段的修改不会导致页面 reflow ，故常用于 DOM 操作的性能优化

基本类型

txt

步骤：

  • 将模板放入缓存
  • 取出模板，字符串格式
  • 字符串格式模板转为node 格式
  • node格式模板加入DOM tree

或者将模板写入 TEXTAREA 标签下

html2node

  • 正则找到第一个标签名
  • 根据标签名，生成一个元素。li–> ul tr–>tbody 等， 一般生成 div
  • 将 html 字符串作为 innerHTML 放入 div 中
  • 判断 div 中的 children 元素有几个？一个的话返回children 元素，多个的话返回 div 元素本身

ntp

_$getNodeTemplate
判断
-如果取出来的格式是字符串的话，执行 html2node 返回 node 格式，并缓存起来，以便下次直接调用

jsx

符合jst语法的模板，例如服务器端模板 freemarkder, verlocity 等

模板解析
采用 TrimPath 库来解析
